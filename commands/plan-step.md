# Plan Step

Create detailed planning workspace for a specific step from `.prove/TASK_PLAN.md`.

**Step Reference**: $ARGUMENTS

## Workflow

Load and execute the plan-step skill (`skills/plan-step/SKILL.md` from the workflow plugin). Follow all sections defined there.

The step reference can be any format: "1", "2.1", "Step 3", "1.2.3". The skill handles parsing, workspace creation, interactive planning, and the readiness checklist.
